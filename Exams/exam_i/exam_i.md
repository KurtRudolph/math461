# Exam I Review

## Combinatorial Analysis

### Binomial Coefficient
\[{n \choose i} = \frac{ n!}{ (n - i)! i!}\]

### Binomial Theorem
\[(x + y)^n = \sum\limits_{i = 0}^{n}{ {n \choose i} x^i y^{n-i}}\]

### Multinomial Coefficient
\[{n \choose n_1, n_2, \dots, n_r} = \frac{ n!}{ n_1! n_2! \cdots n_r!}\]
